FreshMart Documentation

FreshMart is a supermarket / weekly-offers HTML template built with Tailwind CSS (CDN) and vanilla JavaScript. There is no build step — every page opens directly in a browser.

File Structure

├── *.html               11 pages, see "Pages Reference"
├── assets/

Getting Started

  1. Open index.html in any modern browser — no server or build tools needed.
  2. An internet connection is required on first load (fonts, icons and JS libraries are served from CDNs).
  3. To publish, upload the entire template/ folder to any static host.

Shared Components

Every page includes empty placeholder elements that assets/js/components.js fills in on DOMContentLoaded, so header/footer/menu markup exists in exactly one place:

  • <header id="site-header"></header> — sticky glassmorphism header, mega menu, search, theme + RTL toggles
  • <footer id="site-footer"></footer> — newsletter band + footer columns
  • Mobile menu panel is appended to <body> automatically
  • <div id="breadcrumb" data-title="…" data-subtitle="…" data-crumbs='[…]'></div> — page hero + breadcrumb trail
  • <div data-cta="app|branch|loyalty"></div> — reusable CTA band, three content variants

The active nav link is highlighted using data-page="filename.html" on <body> — set this on every new page you create.

Editing Data

Weekly deals, categories, branches, testimonials, blog posts and pricing tiers all live as plain arrays in assets/js/data.js (FM_DEALS, FM_CATEGORIES, FM_BRANCHES, FM_TESTIMONIALS, FM_BLOG_POSTS, FM_PRICING, FM_STATS). Edit an array once and every page referencing it updates automatically via assets/js/render.js.

Dark Mode & RTL

Theme and direction are controlled by assets/js/theme.js and persisted in localStorage (fm-theme, fm-dir). A small inline script in every page's <head> applies the saved theme/direction before first paint to avoid a flash of incorrect styling. Brand colors are CSS variables in assets/css/style.css under :root and html.dark.

Animations

All animation wiring lives in assets/js/main.js: preloader, Lenis smooth scroll, GSAP + ScrollTrigger reveals, SplitType text reveal (data-split-text), AOS scroll reveal (data-aos), Typed.js (data-typed='["A","B"]'), CountUp (data-countup="48" data-suffix="+"), GLightbox (.fm-glightbox), Isotope filtering (data-isotope-container + data-filter buttons), magnetic buttons (.fm-magnetic), Swiper (.fm-swiper with a JSON data-swiper-config), and a reusable countdown (data-countdown="ISO_DATE").
